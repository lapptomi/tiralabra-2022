# Ohjelman suoritus
Ohjelman voi suorittaa antamalla komennon ``poetry run python3 src/index.py`` projektin juuressa.  
Tämän jälkeen projektin juurikansiion luodaan generoitu MIDI-tiedosto ``output.mid``, joka sisältää automaattisesti tuotetun musiikin, jonka voi toistaa erilaisilla DAW-ohjelmilla, tai esimerkiksi verkossa, kuten [Online Sequencerilla](https://onlinesequencer.net/import).
